shinyGoogleCharts

shinyGoogleCharts provides lightweight functions for displaying and interactively editing Google Charts in a Shiny app.

To install, run

if (!require("devtools"))
  install.packages("devtools")
devtools::install_github("shinyGoogleCharts", "mul118")

The shinyGoogleCharts source code is hosted at https://github.com/mul118/shinyGoogleCharts.
