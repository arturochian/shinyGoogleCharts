Shiny Chart Editor output element

Usage

googleChartEditor(inputId, target, type = "Table", options = list(), label = "Edit Chart")

Arguments

inputId
id of the chart editor, available as an input variable
targetId
id of the renderGoogleChart object modified by the editor
type
initial type of the target chart. Defaults to 'Table'
options
initial options of the target chart
label
label for the Chart Editor button

Description

Display a Chart Editor button within an application page. Displays a GUI allowing user to modify properties of the target chart.
